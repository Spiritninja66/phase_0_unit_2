Enumerable#cycle Using cycle would allow you to call the block for each element repeatedly for as many times as you want or forever if the amount of cycles are not specified. A basic example of this is: numbers= [“dog”, “cat”, “me”, “you”] numbers.cycle{ |x| puts x } # prints (“dog”, “cat”, “me”, “you”) over infinitely. You can specify how many times this array can be called like this. numbers= [“dog”, “cat”, “me”, “you”] numbers.cycle(5) { |x| puts x } # prints (“dog”, “cat”, “me”, “you”) #exactly 5 times and then stops.
